# Attribution

## Dataset — Stack Exchange content (CC BY-SA)

The T-bench dataset (`data/`) is synthetic and generated by Compresh, with one
exception: some of the **distractor** turns — the irrelevant filler context used
to test retrieval under load — incorporate text excerpted from public **Stack
Exchange** posts.

Stack Exchange user contributions are licensed under **Creative Commons
Attribution-ShareAlike 4.0 International (CC BY-SA 4.0)** (older posts: CC BY-SA
3.0 / 2.5). Content © the respective authors. Source: https://stackexchange.com

Because the dataset incorporates this CC BY-SA material, **the T-bench dataset as
a whole is released under CC BY-SA 4.0**: you may use, share, and adapt it with
attribution, and any redistributed adaptation must carry the same license.

This applies to the **dataset only**. The T-bench **code** (harness, scorer,
generator) is licensed separately under the **MIT License** — see `LICENSE`.

## Roadmap — removing the share-alike dependency

A future release regenerates the distractors with template-only content (fully
original, no third-party corpora), which removes the share-alike obligation. This
is folded into the next benchmark cycle (TUL 2.1). Until then we publish the set
exactly as measured, so the reported numbers reproduce from the download.
